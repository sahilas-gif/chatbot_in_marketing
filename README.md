# AI Chatbot (Gemini API)

A fully responsive AI chatbot built with HTML, CSS, and Vanilla JavaScript. It uses the Google Gemini 1.5 Flash model to generate responses.

## Features
- Real-time AI responses using Google Gemini API
- Mobile-responsive design
- Modern UI with smooth animations
- Image analysis capabilities

## How to Run
1. Clone the repo.
2. Open `script.js` and add your Google API Key.
3. Open `index.html` in your browser.

## Tech Stack
- HTML5, CSS3, JavaScript (ES6+)
